# TaazaKhabar (News app build in Android studio using Java)
-> Implemented Rapid API to retrieve News Data and integrated Volley Library to fetch data from the API efficiently.<br>
-> Developed a Recycler View to display 100 News Items categorically across 6 different fragments for seamless browsing.<br>
-> Integrated Tab Layout to allow smooth switching between different News categories and Web View to enable users to read News 
   articles in detail.<br>
-> Designed an eye-catching Splash Screen to enhance user experience when the app is launched.<br>
